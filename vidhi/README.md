# Vidhi (DHTI Recipe)

 This directory contains scripts and instructions to build and test DHTI recipes (Vidhi).

 ## Prerequisites

 - Install [Docker](https://docs.docker.com/get-docker/) and [nodejs](https://nodejs.org/en/download/).


## Vidhi Recipes


### Simple Chatbot (sCHAT)

See [chatbot.sh](./chatbot.sh) for building and testing instructions.

### Simple Rag (sRAG)

See [rag.sh](./rag.sh) for building and testing instructions.

### DICOM Imaging Report

See [dicom.sh](./dicom.sh) for building and testing instructions.

### Docktor (Pipelines using MCP server integrator MCPX)

See [docktor.sh](./docktor.sh) for building and testing instructions.
